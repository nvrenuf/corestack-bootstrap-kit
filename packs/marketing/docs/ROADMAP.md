# Roadmap

## Phase 0: hello-web validation (this release)

- Provide minimal pack scaffold with nginx `hello-web`.
- Validate Corestack pack runtime integration and lifecycle commands.

## Phase 1: add pack-local n8n + workflows

- Add n8n service definition in pack compose.
- Add starter workflows in `n8n/workflows`.

## Phase 2: AI provider integration (OpenRouter keys, etc.)

- Introduce provider configuration wiring.
- Add key management and runtime configuration docs.

## Phase 3: publish pipeline (SurviveTheAI-style)

- Define release packaging workflow.
- Add publishing automation and versioned artifact flow.
